== Description ==

If a subscriber is logged in then link to users profile is renamed to a more friendly text and the link is redirected to there profile page.

== Installation ==

1. Unzip the archive of the plugin.
2. Upload folder `wp_profile_link_renamed_relinked` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

Q1. Do I need to edit any files?

A1. No, just activate the plugin and it should work.
